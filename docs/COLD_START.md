按 spec §19 + Plan 4 §F4 执行。本文档既是模板库，也是发送追踪表。

Production URL: https://anxiety-decoder.vercel.app

## Tier 1 · 私信模板

原则：朋友→朋友，2 句话，不解释、不推销。Don't pitch. Don't explain.

### 版本 A · 极简（最熟的朋友 / 已经知道你在搞产品的人）

> 做了个小东西，写给"明明该开始却动不了"的时候用。30 秒能试完，你看看：
> https://anxiety-decoder.vercel.app

### 版本 B · 借口型（不那么熟的朋友 / 怕显得在求 feedback 的）

> 帮我看眼这个？做了一周，想测一下普通人能不能 30 秒上手。
> https://anxiety-decoder.vercel.app

**分配建议**：10-15 名单里 70% 用 A，30% 用 B。按你跟对方关系定，不要群发同一句。

**不要加 disclaimer**（"如果不方便就不用"之类）—— 表面礼貌但增加对方决策成本，等于明示"我在打扰你"。

## Tier 2 · 公开帖

原则：insight first, product last。

### 长版（~280 字，小红书 / 公众号）

> 这两天发现一件事：让我没法开始一项任务的，多半根本不是这项任务本身。
>
> 比如我已经一周没改简历了。打开 Word，盯一眼，关掉。我以为是"懒"或"拖延症"。
>
> 但仔细问自己——"你具体在怕什么？"
>
> 答案突然变得很碎：
> · 怕投了被拒
> · 怕没投错过窗口期
> · 怕同期同学已经在推进
> · 怕 HR 觉得我项目经历不够亮
> · 怕……其实我不知道
>
> 真正"要做"的事只有一件——把现在的 PDF 打开，改一段。但脑子里挤满了想象出来的灾难、跟同学的对比、关于自己能力的判断。这些东西伪装成"事情"，让我以为我有 5 个任务要做。其实只有 1 个。
>
> 所以这周做了一个工具。它问你一句："你现在到底在回避什么？"然后帮你把脑子里那一坨拆成三类：真正要做的、想象出来的最坏情况、纯粹的雾。
>
> 30 秒可以试一下：https://anxiety-decoder.vercel.app

**配图建议**：用 `/card/:id` 截图当封面（Phase 1 readOnly 卡片就是为这个设计的）。封面 + 1-2 张产品截图（landing 输入框 + decode card）即可。

### 短版（~120 字，即刻）

> 一个发现：让我没法开始一项任务的，多半不是任务本身。
>
> 比如改简历。我以为是"我懒"，但问自己"你具体在怕什么"——答案是一坨：怕被拒、怕错过窗口、怕同学已经投了、怕……其实不知道。
>
> 真要做的事只有一件，打开 Word 改一段。但脑子里挤满了想象出来的灾难，让我以为有 5 件事要做。
>
> 做了个工具帮自己拆开这一坨：https://anxiety-decoder.vercel.app

## Week 4 量化目标（spec §19.4）

| 指标 | 目标 | 实际 |
|---|---|---|
| 真实用户数（不含你 / 不含同设备） | 20-30 | |
| 总解码次数 | 50-80 | |
| ≥2 次解码用户 | ≥5 | |
| opt-in 邮箱 | ≥6 | |
| 已响应验证 | ≥3 | |

**Phase 5 启动条件**：5 项指标至少 **3 项达标** → 视为冷启动证明产品命题站得住，可正式排定 Phase 5（迭代优先级由"哪一项指标最弱"决定）。
- 5 项全中 → 写 retrospective 进本文档底部，进 Phase 5
- 3-4 项达标 → Phase 5 优先级聚焦在未达标的那 1-2 项；retrospective 必须明写"为什么这项没达标"
- ≤ 2 项达标 → 不进 Phase 5，回到 §1.2 命题层重新审视产品定位（可能需要换用户群 / 换价值主张）

**未中项的对应卡点诊断**（spec §19.4 漏斗）：
- Landing 提交率低 → F2 视觉 / Landing copy 再调
- 解码完成率低 → AI prompt 调优（F1）
- 留存低 → 核心体验调（result 页 / EvidenceStats 还需打磨）
- opt-in 率低 → EmailOptIn UI / 文案调

## 道德底线（spec §19.5）

- **不**反查朋友在 DB 里的内容。永远不。
- **不**催 Tier 1 给反馈。
- 如果某个 Tier 1 反馈"产品让我感觉更糟"，**立即停止**对该联系人的邀请，修问题，修完之前不再触达。
